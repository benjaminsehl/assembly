# Skills

This directory contains the public command-like skills exposed by the plugin:

- `project-status`
- `product-discovery`
- `prototype`
- `spec`
- `plan`
- `build`
- `test`
- `qa`
- `review`
- `code-simplify`
- `ship`

Detailed workflow guidance lives in `references/`, not in triggerable support skills. This keeps the public surface small enough for reliable routing and avoids conflicts with users' existing skills.
