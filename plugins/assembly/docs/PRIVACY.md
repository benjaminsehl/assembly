# Privacy

Assembly is a local agent plugin made of skills, references, templates, and helper scripts. It runs inside Codex or Claude Code; the same bundle works in both.

The plugin does not add telemetry, network services, analytics, or data collection. Any files it reads or writes are the result of normal agent tool use in the user's local workspace.

Users are responsible for reviewing agent actions before allowing sensitive, destructive, external-message, credential, or money-moving work.
